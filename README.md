## Rules

- **CLEAR** — GATEを通過してホールイン
- **SUCCESS** — GATEを通過
- **MISS** — GATEを未通過 ／ 球に接触

## Controls

**Ball**

- **SHOT** — 左ドラッグ（球の近くで離すとキャンセル）
- **CHANGE** — ←→ ／ A D

**Camera**

- **ROTATE** — 右ドラッグ
- **ZOOM** — ホイール

## Features

- **STAGES** — 全4ステージ
- **RECORD** — ベスト打数＆タイムを記録
- **CONFIG** — 音量・カメラ感度の設定に対応
